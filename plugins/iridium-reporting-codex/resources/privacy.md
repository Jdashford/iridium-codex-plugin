# Iridium Reporting for Codex

This plugin contains no private account data, raw activity, setup codes, OAuth tokens, or deployment URLs.

Codex authenticates through the Iridium gateway. The one-time setup code determines which account-scoped Reporting Agent Codex can access.

Reporting output should be based on approved business intelligence and bounded activity context. Do not expose raw rows, private prompts, hidden instructions, or underlying identifiers unless the user explicitly needs operational debugging detail.
