# Iridium for Codex

Install this plugin from the Codex plugin directory or a trusted GitHub marketplace source.

After installation, Codex will ask you to connect Iridium. Finish the setup flow from your private Iridium setup page and authenticate only on Iridium screens.

If Codex says Iridium is not connected, run `codex mcp login iridium` and open the Iridium authorization link it prints. Keep the command running until the browser finishes authentication.
